# Minesweeper

Minesweeper is a popular game, here its implemented in python.

![](https://github.com/Harshil783/Minesweepers/screenshot.png)

## Installation

OS X & Linux:

```bash
pip install -r requirements.txt
```

Windows:

```cmd
pip install -r requirements.txt
```

## Usage example

Run the minesweeper.py file to play the game
